animate

Muss in Subklassen überschrieben werden, falls man das Objekt animieren will. Wird im default ca. 60 Mal pro Sekunde aufgerufen. Änderungen der Parameter wie etwa x und y werden so direkt angezeigt, wenn man sie überschreibt. Alternativ kann man auch bestehenden Objekten neue Animationen zuweisen.
